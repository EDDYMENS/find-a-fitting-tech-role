Data Scientist

In this day and age software products house a lot of data, from user data to data about how customers are interacting with the product itself.

The job of the data scientist is to use math knowledge and software tools to make meaning of this data.

Meaning here can be anything, whatever information a company wants to know about their product a data scientist can help make that possible.

Sometimes the data used might not be directly correlated to what the company is trying to find out, but these professionals can create a link and provide the information that is sort after.

Coding

You will need to know coding to excel in this role, it won't be the kind needed to build a full fledged software application but the kind needed to work with data and to translate mathematical models to scripts.

Working in teams

Wether you are team dependent will be based on the company you find yourself in. You might find yourself in a team of professionals who work with data or all alone only called on when needed.

Working alone

Most companies only have a single data scientist, which means you might find yourself working alone and only presenting your results to a team when needed.

Impact

Data scientist are able to bring to light things that are not obvious within a software product through data and this is powerful.

Managing a team

If you do find yourself in a data department you might end up having to managing it.

Dealing with customers

For the most part data scientists don't interact much with end customers.

Learning Curve

You will need to get familiar with a programming language such as python as well as some software tools like Tableau. Also having math skills will be very helpful.
